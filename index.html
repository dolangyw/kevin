Tentang Saya

Saya Kevin Fernando, full-stack developer dan pembuat bot yang fokus pada pengalaman pengguna, otomasi, dan sistem berbasis permainan. Saya membangun solusi end-to-end: dari antarmuka web interaktif, backend yang terintegrasi dengan database real-time, hingga sistem game dan plugin server. Saya sering bereksperimen dengan UI minimalis dan estetika, serta membuat konten singkat seperti lirik dan video edukasi untuk platform seperti TikTok.

Portofolio & Proyek Terpilih

Sistem Integrasi Database & Dashboard Real-time

Bot Otomatisasi WhatsApp

Bot dengan fitur pengunduh media, reply-based actions (kirim otomatis saat reply), handling presence/offline, dan integrasi utilitas seperti image/audio processing.

Terminal Web Interaktif

Terminal simulasi untuk eksplorasi fitur, command-line UI, dan demonstrasi automasi bot dalam tampilan terminal browser.

RPG (MonoGame / C#)

Game RPG bergaya Stardew-like: sistem bertani, memancing, menambang, NPC dengan story & sub-story, quest, dan progression yang terstruktur.

Roblox, HUD & Mekanik Gameplay

Server Minecraft & Plugin

Pengembangan plugin Paper dengan sistem ekonomi, crate/gacha 7 tier, GUI manajemen toko, transaksi (SELL/BUY/BARTER/GAMBLE), dan fitur party/roleplay.

Website Pemutar Musik & Lirik
