Tentang Saya

Saya Kevin Fernando full-stack developer dan pembuat bot yang fokus pada pengalaman pengguna, otomasi, dan sistem berbasis permainan. Saya mengerjakan proyek web interaktif (player lirik seperti Spotify, terminal web), bot chat dan automasi (Baileys / whiskeysocket), serta pengembangan game dan server game. Selain itu saya juga fokus pada integrasi database real-time, arsitektur backend sederhana tapi andal, dan sistem RPG/Minecraft dengan ekonomi, gacha, dan GUI khusus. Selain programming, saya suka estetika dan fotografi, sering membuat konten lirik/video pendek untuk TikTok, dan suka bereksperimen dengan UI minimalis.
